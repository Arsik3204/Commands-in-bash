Специальные символы

* - используется для того чтобы показать либо отсутствие символов, либо все символы сразу

[] - соотвествуют любым символам заключенным между ними

\ - Escape symbols, используется для исключения символов,которые идут перед ним

tree . - появляется так называемое дерево

Команды для управления процессами
ps - отображает список процессов в текущей сессии
ps -f - отображает список процессов в расширенном формате
ps -e - абсолютно все процессы
ps -u[username] - отображает процессы определенного пользователя
ps -p[PID] -  отображает информацию о процессе с этим [id]

Команда KILL

Ctrl + c - останавливает процесс
kill [PID] - позволяет убить процесс только нужно указать его [id]
kill -9 - убивает процесс окончательно 

Отправить процесс в фон

command & - вводишь комманду которую хочешь оставить в фоне о обязательно этот символ & 
jobs - чтобы увидеть какие процессы работают в фоне
fg [n] - эта команда позволяет вернуть процесс на передний план (обязательно нужно указать номер процесса)
